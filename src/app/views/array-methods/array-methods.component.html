Methods:

  1. push -- adds an item at the end of the array
  2. pop -- removes an item from the end of the array
  3. shift -- adds an item at the start of the array
  4. unshift -- removes an item from the start of the array
  5. slice(1,4) -- return the value(does not change the array) item from 1 to 4(exclusive)
  6. splice(1,4,'f') -- replace 4 items from position 1 with 'f'
  7. map-- modify each item of the array
  8. filter-- filter items on certain condition
  9. reduce-- reduce to compute something
